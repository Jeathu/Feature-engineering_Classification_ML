# 1.Création de environnement virtuel

    (windows)
        - python -m venv venv
        - venv\Scripts\activate
        - python -m pip install --upgrade 
        - pip numpy pandas matplotlib jupyter ipykernel scikit-learn


    (Mac OS / Linux)
        - python -m venv venv
        - venv/bin/activate
        - python -m pip install --upgrade pip numpy pandas matplotlib jupyter ipykernel scikit-learn


## Pour créer u fichier requairements.txt et organiser les packages
    pip freeze > requirements.txt



## Si vous avez une erreur :
     Running cells with 'venv(python)' requires the ipykernel package to be installed.
    
   - Etape à suivre:
        - Il faut supprimer le venv 
        - Il faut réinstaller le venv et suivre les étape de instalation
